# AGENTS 指南

## 写作约定

- 使用汉语表达概念术语，英语作为实现别名；广泛被采纳的业界通用术语除外；高频术语列入术语表。
- 避免使用“不是——也不是——而是——”的句式；优先使用正面定义，而不是借连续否定增强说服力。
- 避免使用大量的顿号（、）串联示例；优先总结归纳出特征，而不是借举例逃避下结论。

## 迭代约定

- 使用 mise 管理工具链，新 worktree 用 `mise trust` 和 `mise install` 初始化环境。
- 启动本地服务前先检查目标端口（任一本地地址）；若已有非本会话服务监听，则优先复用，不另启同端口；确需新服务时换端口并告知用户。
- 仅在用户要求时提交，commit message 使用 `type: 简短中文说明`，如 `feat: 添加深色模式主题`。
- 若 Astro 开发服务由 agent 启动，默认保留供用户确认；提交完成后停止该服务；用户启动的服务不处理。
