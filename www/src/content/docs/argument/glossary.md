**项目真相**

项目真相，是当前项目实际依照的内容。它决定项目现在如何运行、如何被理解，后续修改和判断时必须参考什么，也为不同任务场景生成投影视图提供依据。

**术语表**

术语表用于维护项目中的稳定词汇。它让人、LLM 和 agent 在同一语义空间中工作，减少同义词、近义词、行业俗称和错误写法带来的歧义。

**工作语言**

工作语言，是项目在理解、命名、判断、记录和维护业务语义时实际使用，并且能够被主要维护者持续复用的自然语言。它由业务概念、业务规则、客户口径、产品取舍和日常记录中的稳定用词共同形成。

**投影视图**

投影视图，是面向某次任务、某个 agent 执行场景、某次审计、业务理解、产品调研或管理查看，从项目真相中抽取、重组和呈现出来的一部分内容。

**领域**

领域，是项目真相的组织单位。它把同一范围内需要一起理解、修改、审查和持续维护的代码实现、术语、约束、规则、决策记录和必要的历史记忆组织起来。
