**專案真相**

專案真相，是目前專案實際依照的內容。它決定專案現在如何執行、如何被理解，後續修改和判斷時必須參考什麼，也為不同任務場景生成投影視圖提供依據。

**術語表**

術語表用於維護專案中的穩定詞彙。它讓人、LLM 和 agent 在同一語義空間中工作，減少同義詞、近義詞、業界俗稱和錯誤寫法帶來的歧義。

**投影視圖**

投影視圖，是針對某次任務、某個 agent 執行場景、某次審計、業務理解、產品調研或管理檢視，從專案真相中抽取、重組並呈現的一部分內容。

**領域**

領域，是專案真相的組織單位。它把同一範圍內需要一起理解、修改、審查和持續維護的程式碼實作、術語、約束、規則、決策紀錄和必要的歷史記憶組織起來。
